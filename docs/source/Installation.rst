Installation
============

pyht requires Python ``>=3.8`` and can be installed from pypi via:

.. code-block:: bash

   python -m pip install pyht


To install directly from github, you can run:

.. code-block:: bash

   python -m pip install git+https://github.com/playht/pyht.git

For development, its recommended to use Poetry:

.. code-block:: bash

   git clone https://github.com/playht/pyht.git
   cd pyht
   poetry install
